# Preface
NexuraTube is a free and open-source project. If you wish to contribute, please follow the guidelines below.

## Bug Reports and Feature Requests
Please submit any bugs or feature requests directly through our official [GitHub Repository](https://github.com/Mashalmazen/Nexuratube/issues).

## Pull Requests
1. Fork the project.
2. Create a new branch for your feature or fix.
3. Submit a pull request with a clear description of your changes.

## AI Contribution Policy
Generative AI may be used for contributions as long as the contributor takes full responsibility for the code and ensures compliance with the project's license.
